# Image Editor Web App

A simple **web-based image editor** that lets you apply real-time filters to images, use preset effects, and download the edited image. Built using **HTML, CSS, and JavaScript** with `<canvas>` for image rendering.

---

## Features

- Upload any image and edit it in real-time.
- Adjustable filter sliders:
  - Brightness
  - Contrast
  - Saturation
  - Hue Rotation
  - Grayscale
  - Blur
  - Sepia
  - Opacity
  - Invert
- Preset effects:
  - Drama
  - Vintage
  - Old Money
  - Pop
  - BW (Black & White)
  - Cool
  - Warm
  - Soft
  - Moody
- Reset filters to default.
- Download the edited image.

## How to Use

1. Open `index.html` in a web browser.
2. Click **Choose File** to upload an image.
3. Adjust the **filter sliders** to apply custom effects.
4. Click on **Preset buttons** to instantly apply predefined styles.
5. Click **Reset** to restore all filter values to default.
6. Click **Download** to save the edited image.
